## 第一讲
极限一节的考察形式：

等价无穷小的证明。在证明时，遇到函数作差
```
x - sin(x)
-------------------
arcsin(x)-arctan(x)
```
应当注意使用泰勒展开式（不需要全部展开，常常只需要前两、三项）。

而变限积分，应当注意使用洛必达法则。
### 洛必达法则的重要结论
若 x→0 时有 f(x) 与 g(x) 等价，则 x→0 时，f(x) 与 g(x) 的变上限积分等价，f'(x) 与 g'(x) 等价。
> 完整详细的语句有待验证。
