## 基本数值类型
### int 类型
不同于其他语言，Python 只有一种整型，且可以存储任意大的数字并进行运算。

支持四种进制：2、8、10、16

`0b`：二进制

`0o`：八进制

`0x`和`0X`：十六进制

输入的时候这样标记，但是输出的时候并不是。

输出的时候仍然依赖于特定的格式控制字符串。

### float 浮点类型
Python 只有一种浮点类型。

浮点格式是
```
(一个绝对值小于10的实数)E(整数)
```

表示
```
实数x10^(整数)
```
### Python 特有的复数类型
Python 使用 j/J 表示虚数单位 sqrt(-1)
```python
(1+2j) + (3+4j)
```

返回`(4+6j)`。

Python 有复数运算的相关方法，课下自己了解一下。
## 非数值类型
### 串
Python 中所有的字符都是字符串，不存在 char 类型。

用`'<串值>'`或`"<串值>"`括起来的是不可跨行的单行字符串。

用三引号`'''<串值>'''`或`"""<串值>"""`括起来的，可以跨行，且记录换行符`\n`。

引入两种不同的引号，可以避免在字符串中包含引号时不得不转义，破坏可读性的情况。
#### 串的访问
string[index] 访问索引为 index 的串中字符，索引从 0 开始。

字符串是常量（不可变类型），可读不可写。
#### 阻止串转义
```python
s = r"C:\Users\Administrator\Desktop"
```
这个字符串前面的`r`阻止了字符串中的反斜杠被解释为转义字符，

保证了文件路径不发生错误。

在这种情况下调用`len()`函数求字符串长度时，反斜杠也会当做一个普通字符被计数。
## Python 的内置函数
### eval()
*去掉字符串外层的引号。*

事实上它的效果并没有“对字符串求值”那么复杂。

因此你甚至可以拼接表达式。

```python
a = "print("
b = "'this is the output'"
c = ")"
d = a + b +c
print(d)
eval(d)
```
输出
```
print('this is the output')
this is the output
```
## Python 特有的四种复合数据类型
### 概述
list 列表，可变的多种元素组合类型

`[元素1, 元素2, 元素3]`

tuple 元组，不可变的多种元素组合类型

`(元素1, 元素2, 元素3)`

在基于数据表的数据库（关系数据库）中，

一横行称为一条记录，或称为一个元组。

Python “元组”的术语可能是从此处来的。

dict 字典

set 集合，数学意义上的集合，具有元素无序、元素唯一两条特性。

这两种都用花括号

`{内容}`

表示，但有所区别。
### list 列表
用方括号括起来，可以存储各种类型元素的**有序**类型。

可以下标访问，可以排序。
### tuple 元组
(元素1, <元素2>, <元素3>)

**任何一组以逗号分隔的对象，省略括号时，会被默认为元组。**

如果省略元素1后面的逗号，不会产生元组，而是产生一个以元素1的类型为类型的变量。
### dict 字典（重点）
字典是用大括号{}之间，用逗号分隔的元素集合。

其元素由 key: value 格式的“键-值”对来存储，

无序存放，不允许使用下标访问，必须通过键来访问。
```python
>>> s={'x':3,'y':6}
>>> s
{'x': 3, 'y': 6}
>>> type(s)
<class 'dict'>
>>> s['x']
3
```
键可以是任何有序的类型，字符串、整数和列表都是可以的。

但是一般使用字符串。

一对大括号{}，不包含任何元素，也会被默认为字典。
### set 集合
Python 允许用`{}`或集合类型的构造函数`set()`来生成集合。

```python
a = {元素1,元素2,元素3,元素3,元素4,元素5}
```

如果这样生成元素，重复的元素会被丢弃，只保留一个：

```{元素1,元素2,元素3,元素4,元素5}```

空集合就用不带参数的构造函数 set() 来产生。
## 常用的 Python 函数
### 来自模块的函数
#### 导入其他模块的方法
使用时导入到程序内：
```python
import <模块名>
```
这样导入，之后用到模块内函数需要注明模块名：
```python
math.sqrt()
```
如果不考虑函数名的互相混淆问题，可以这样导入模块：
```python
from <模块名> import <函数名，或通配符*>
```
这样可以直接写`sqrt()`，就像你自己在程序内声明了这个函数一样。

但是一旦出现两个库有重名函数，这样就要报错。

调用方法时，仍然建议写出方法来自的库的名字。
#### 常用的模块
math: 数学模块

cmath: 复数计算模块

random: 随机数模块

time: 时间模块

calendar: 日历模块
### math 模块
包含常数：

math.pi math.e

包含函数：
fabs() 求浮点绝对值。

log(真数, 底数)

求对数，若省略底数，默认为 e（即自然对数）。

ceil(), floor() 向正无穷 / 负无穷取整。

fmod(x, y)

求 x/y 之余数，返回值为浮点数。

degrees() 从弧度转为角度。

radians() 从角度转为弧度。

asin() acos() atan() 反三角函数。去掉开头的 a 就是对应的三角函数。
### cmath 模块
**函数名和 math 相同**，但是 cmath 能对复数做运算。

如果和 math 同时调用，务必区分从哪个库调用出函数来。

### random 模块
choice(列表) 从列表中随机抽取一个元素。

sample(列表, 数量) 从列表中随机抽取`数量`个元素，**组成一个新列表**。

shuffle(列表) 将列表打乱顺序排列。
### time 模块
time() 返回时间戳。从1970.01.01.00.00.00 UTC 开始的秒数，不考虑闰秒。

除此之外，还有时间的格式化功能。
### calendar 模块
上课略去不讲，自己下来自学。
### 系统内置函数
range()

产生一个迭代器对象，每次这个对象被调用，就返回自身的下一个数值。

range(左端点, 右端点, 步长)

产生从左端点（包含）到右端点（**不包含**）区间，给定步长的迭代对象。

略去左端点，则左端点的默认值为 0。

略去步长，则步长的默认值为 1。

abs()

返回绝对值。若参数是复数，返回 float 类型的模。
## 常用的运算符
### 数值运算
a % b：求 a 除以 b 的余数。

如果 a, b 是浮点数，也能按照期望去运算，但是所得的数值可能不精确，比如有0.0000001数量级的误差。

这个是由于计算机的字长有限，不能精确运算导致的。

a // b：求 a 整除 b 的商。

以上运算都支持负数。

但是对于整除，(-7)//(-4)==-2，

很有可能是 floor() 取整的原因。
### 位运算
计算机使用补码来运算、使用补码来存储运算结果。

但是输出到屏幕上给人看时，Python 使用原码。
#### 复习二进制编码
正数：原反补码相等。

负数：

原码符号位不变，其余位取反→反码。

反码+1→补码。
